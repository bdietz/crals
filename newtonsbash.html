More bash commands!

(Nearly) every command in bash has an equal and opposite command. Here are some we have found.
cut -- cut out selected portions of each line of a file paste -- merge corresponding or subsequent lines of files
diff -- compare files line by line patch -- apply a diff file to an original
unix2dos -- DOS/Mac to Unix and vice versa text file format converter dos2unix -- DOS/Mac to Unix and vice versa text file format converter
head -- display first lines of a file tail -- display the last part of a file
cat -- concatenate and print files tac -- concatenate and print files in reverse

This awesome list was inspired by a Let's Do This a Thon at BitCurator 2018!
